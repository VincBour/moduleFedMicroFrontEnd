## `Installation`

### `database`

In the project directory run :

docker-compose -f docker-compose.yml up --build

### `Backend`

In the folder backend run: 

1. yarn
2. yarn start

the backend in [http://localhos:8080](http://localhost:8080)

### `Frontend`

In the folder frontrend run: 

1. yarn 
2. yarn start
